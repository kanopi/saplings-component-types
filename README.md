# Saplings - Components

Creates Saplings components.

## Features

* Creates the following Paragraphs components:
  * Accordion/Accordion Item
  * Block
  * Card
  * Card (Existing)
  * Carousel/Carousel Item
  * Columns
  * FAQ/FAQ Item
  * Filtered List
  * Hero
  * Media
  * Media Collection
  * Side by Side
  * Statistics
  * Tab/Tabs
  * Text
